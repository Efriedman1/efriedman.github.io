Roam

iOS App


The first application I built and published to the iOS app store while attending Make School.

Roam was based on the concept of choice overload, which can simply be defined as providing the user with an overwhelming amount of choices. Roam addresses this issue by providing you with a very minimal list of choices related to your desired category to allow you to quickly and efficiently make a decision with little to no bias. Don't like any of the options? Simply generate a new list!

Stop wasting time figuring out where to go and start Roaming!

The first application I built and published to the iOS app store while attending Make School.

Tutorial Project

HTML CSS & JavaScript Web App


A tutorial project I built for CIS 3344 (Client Side Web Scripting) in which I created a reusable JavaScript component and a tutorial on how the component was built and how to implement it in your code.

The component is a drop down menu that allows you to filter an array of objects. The dropdown menu has a search bar that allows you to filter the menu items as well. The menu is highly customizable and can have as many filterable values and filter as large of a list as the user wants.

In addition to the drop down menu the code features a CSS grid for the object list and a hover overlay effect.
